# Informatikai Projektfeladat 2026 - SZA / BA

🔶 Ez a projekt egy CodeIgniter4 (PHP) keretrendszer alapú webprojekt / webapp lenne. A projekt pontos célja / tartalma még nem meghatározott, de hamarosan ez nem így lesz. 

## ✅ Használat
1. 📥 Letöltöd a teljes projektet manuálisan vagy:
```bash
git clone https://github.com/Drewsheep/IP-CI4-SZA-BA.git
```
2. 📦 Telepíted a függőségeket (Composer-t le kell külön töltened):
```sh
composer install
```

3. 🎉 Használod az appot!
